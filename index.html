Setting Up the Document

Example Code

<!DOCTYPE html>
                        <html>
                        <head>
                        <title>Page Title</title>
                        </head>
                        <body>
                        
                        <h1>This is a Heading</h1>
                        <p>This is a paragraph.</p>
                        
                        </body>
                        </html>

Explanation

This is the setup of a basic HTML document. Remember, that you need to import your stylesheet, and you can also break up the sections with tags like <main>, <header>, <footer>, <article> etc.
